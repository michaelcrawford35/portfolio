Maitre D Tipping App

A small app developed with JavaScript that allows the user to enter the bill total and their tipping percentage.
The app then calculates tip, tax and displays a total to the user.

Volume of a Sphere

Another small app developed using HTML Forms. The goal of this project was to use HTML Forms to have interaction with the page.
The end goal is to allow the user to enter the radius and calculate the result.

LEFT 4 DOOM

LEFT 4 DOOM is a game built with the Unity engine that takes inspiration from Left 4 Dead and DOOM. It is the culmination of approximately 18 straight hours of game development along with a few other hours of previous work on the project.
